## 1) Publicación en Platzi (comentario del Proyecto 2) — ESTA es tu entrega

🔶 **SUELDAZO // MAGI — Sistema de análisis salarial tech de LatAm**

**Qué es:** una terminal (con estética NERV/MAGI de *Neon Genesis Evangelion*) que te muestra **al instante la mediana real de sueldos tech** de LatAm por rol, seniority y país — antes de pedirte nada. Eliges tu perfil y ves la referencia de mercado con su rango P25–P75.

**Para quién:** cualquier persona en tech (dev, data, DevOps, QA, diseño, PM) que quiera saber si está por encima o por debajo del mercado antes de negociar.

**Cómo funciona con datos reales:** las medianas están **agregadas de fuentes públicas 2025/2026** (Talently, Vacantes Digitales, Get on Board, CodersLink) — los tres "nodos MAGI" (MELCHIOR / BALTHASAR / CASPER) representan esas fuentes. Además, cuando transmites tu dato, se guarda de forma **anónima en una base de datos real (Supabase)** — con RLS (solo INSERT, los datos quedan privados) y una función `security definer` para el contador. El sistema muestra tu "patrón" (Azul = bajo umbral, Naranja = en parámetros, Verde = sobre umbral).

**Stack:** HTML + CSS (terminal NERV con clip-path, scanlines CRT, respeta `prefers-reduced-motion`) + JS vanilla + Supabase. 100% vibe coding.
